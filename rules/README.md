# Rules

Outlook rules templates for the GTDo workflow.

The book's approach **prefers native Outlook rules over VBA macros** when possible — they're more portable, more reliable, and don't require code knowledge.

## Available rules

- [`sent-to-waiting.md`](sent-to-waiting.md) — **Sent → Inbox → W (Waiting)**: every sent email becomes a tracked "waiting for" item in your Inbox.
